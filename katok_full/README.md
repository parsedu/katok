# Katok Full - PyQt Projects

Репозиторий содержит различные проекты на PyQt5.

## Структура проекта

### PyQt/full/l1/
Базовые примеры работы с PyQt5:
- `MyWindow.py` - базовый пример создания окна в ООП-стиле
- `ex2.py` - пример использования виджета в диалоге
- `ex3.py` - пример блокирующей операции

### PyQt/Laba1/
Лабораторная работа №1:
- `1zadanie.py` - основное приложение
- `1zadanie_main.py` - альтернативная реализация

### PyQt/LAB2/
Лабораторная работа №2:
- `Zadanie2.py` - работа с очередями, стеками и синхронизацией
- `num4.py` - пример использования модуля queue с потоками

### PyQt/LAB3/
Лабораторная работа №3:
- `3zadanie.py` - работа с окнами (позиционирование, прозрачность, модальность и т.д.)

### PyQt/LAB6/
Лабораторная работа №6:
- `6zadanie.py` - работа с событиями клавиатуры

### PyQt/LAB7/
Лабораторная работа №7:
- `num1.py` - пример фильтрации событий клавиатуры

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск примеров

### Базовые примеры
```bash
cd PyQt/full/l1
python MyWindow.py
python ex2.py
python ex3.py
```

### Лабораторные работы
```bash
cd PyQt/Laba1
python 1zadanie.py

cd PyQt/LAB2
python Zadanie2.py
python num4.py

cd PyQt/LAB3
python 3zadanie.py

cd PyQt/LAB6
python 6zadanie.py

cd PyQt/LAB7
python num1.py
```

## Требования

- Python 3.6+
- PyQt5 >= 5.15.0

## Примечания

- Все примеры используют PyQt5
- Некоторые примеры требуют GUI окружения для запуска
- Код проверен на работоспособность и исправлены основные ошибки
